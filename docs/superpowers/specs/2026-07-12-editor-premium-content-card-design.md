# 편집기 프리미엄 콘텐츠 카드 설계

## 목표

메인 편집창에 네이버 프리미엄콘텐츠 채널을 알리는 단일 홍보 영역을 복원한다. 홍보 영역은 캡처·편집·저장 명령과 경쟁하지 않고, 기존에 정리한 명령바의 한 줄 구조를 유지해야 한다.

## 배치

- 캡처 기록 패널 하단에 고정 카드형 버튼을 둔다.
- 카드 위의 기록 목록만 스크롤되므로 기록이 많아져도 카드는 같은 자리에 유지된다.
- `Narrow` 모드처럼 기록 패널이 접힌 상태에서도 접근할 수 있도록 상단 더보기 메뉴에 `추천 콘텐츠` 항목을 둔다.
- 상단 명령바와 상태바에는 홍보 요소를 추가하지 않는다.

## 시각과 문구

- 네이버 프리미엄콘텐츠를 인식할 수 있는 녹색 포인트와 `N` 마크를 사용하되, 앱의 흰색·회색 표면과 둥근 모서리 체계를 유지한다.
- 헤드라인은 `AI 실전 활용법`, 보조 문구는 `네이버 프리미엄콘텐츠에서 보기`로 고정한다.
- 문구 회전, 번쩍임, 자동 애니메이션은 사용하지 않는다.
- 전체 카드를 하나의 버튼으로 만들어 클릭 영역을 충분히 확보한다.

## 동작과 접근성

- 카드와 더보기 메뉴 항목은 `https://contents.premium.naver.com/market/ai`를 기본 브라우저로 연다.
- 카드에는 도구 설명, 키보드 포커스 표시, 스크린리더용 접근성 이름을 제공한다.
- 링크 열기에 실패하면 앱이 종료되지 않도록 상태 표시줄에 안내 문구를 표시한다.

## 검증

- 링크와 표시 문구는 테스트 가능한 카탈로그에 정의하고 단위 테스트한다.
- 1600px와 1100px 편집창에서 카드가 기록 목록이나 버튼과 겹치지 않는지 시각 검수한다.
- 전체 Release 테스트와 빌드를 실행한다.
- 기존 `v1.3.10` 설치 파일은 보존하고, 변경본은 새 버전 `v1.3.11` 설치 파일로 생성한다.
