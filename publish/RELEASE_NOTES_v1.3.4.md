# 신캡쳐 v1.3.4

> UX 정리 — 광고 배치 개선 + 효과 약한 기능 제거.

## 🔧 변경

### 작은 윈도우에서 광고 배너 위치 변경
편집기 윈도우가 좁을 때(< 1100dip) 툴바의 광고 배너가 사라지면서 빈 공간이 생기던 문제를 정리. 이제 좁은 화면에선 status bar의 **API 키 발급법** 버튼 우측에 compact 배너로 자동 이동.

### v1.3.3에서 추가된 "🪄 배경 제거" 버튼 제거
일반 사각형 캡쳐에 GrabCut(InitWithRect)을 적용하는 기능이었으나, "가장자리=배경" 가정이 일반적인 캡쳐 시나리오에선 자주 깨져 결과 정확도가 낮음. 폴리곤으로 직접 영역을 그리는 **스마트 컷(Ctrl+Shift+G)**이 같은 GrabCut을 더 정확하게 활용하므로 그쪽을 권장.
