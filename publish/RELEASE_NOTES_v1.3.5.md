# 신캡쳐 v1.3.5

> 캡쳐 후 편집창 초기 확대율 정책 개선 릴리즈.

## 변경 사항

### 편집창 초기 이미지 크기 개선
- 캡쳐 이미지가 편집창에 들어갈 때 100%로 표시 가능한 경우 무조건 100%로 표시합니다.
- 이미지가 화면보다 클 때만 자동 축소합니다.
- 자동 축소 하한을 80%로 제한했습니다. 80%로도 화면에 다 담기지 않는 큰 이미지는 80%로 열고 스크롤바를 표시합니다.
- 기존 `FitToView` 흐름을 `ApplyInitialZoom`과 `EditorZoomPolicy`로 분리해 줌 정책 충돌을 줄였습니다.

### 안정성
- 초기 줌 계산 정책에 대한 회귀 테스트를 추가했습니다.
- 편집기 OCR/번역 흐름은 `EditorOcrService` 경유로 정리된 현재 코드 상태를 포함합니다.
