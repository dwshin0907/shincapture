# 신캡쳐 v1.3.3

> 일반 캡쳐 결과에도 배경 제거 — 새 도구 + v1.3.2 자유캡쳐 회귀 hotfix.

## ✨ 신규

### 🪄 배경 제거 버튼 (편집기 툴바)
일반 사각형 캡쳐(Ctrl+Shift+C 등) 결과에도 **GrabCut**으로 배경을 자동 제거. SmartCut(Ctrl+Shift+G)은 폴리곤을 그려야 하지만, 이건 캡쳐 후 버튼 한 번 — 이미지 가장자리=배경, 안쪽=객체로 가정하고 처리. 객체 주변에 약간의 여유가 있는 캡쳐에 효과적.

결과는 **새 history 항목**으로 추가되어 원본도 보존됩니다.

## 🐛 수정

### 자유캡쳐(Ctrl+Shift+F) 회귀
v1.3.2에서 자유캡쳐의 검은 점/노치 fix 시도(boundary 5x5 dilate)가 폴리곤 좁은 부분(곡선/얇은 영역)에서 안쪽까지 침투해 inside 영역이 사라지고 boundary만 보이는 회귀를 만들었음. v1.3.1 시점의 GraphicsPath.SetClip 방식으로 복원. 미세한 검은 점은 손그림 입력의 본질적 한계로 받아들이고, 정상 캡쳐 결과를 우선.
